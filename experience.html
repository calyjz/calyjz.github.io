Experience

Undergraduate Research Assistant

I worked under graduate student Megan Sumner and Dr. Matthew Guzdial on a project called Mechanic Maker. It is a level creation tool which employs an assitive AI agent that aims to make game development more accessible. This project uses Unity, Python, and a bit of C++. I modified the current rule-based model of the AI agent by expanding the number of engines the model returns in the backend. I also created a database of rulesets taken from the user study. The next step is to let the model use the rules in the database as well as the rules from the current session to learn from. Additionally, the dataset will also be used to train and test various other models.
